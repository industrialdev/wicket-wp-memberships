# Switch a Membership to a Different Tier

Switching a membership moves a member from their current tier to a different one — for example, an upgrade or a downgrade. The member keeps their existing end date; only the tier changes. A new membership is created on the chosen tier and the original is cancelled.

## Before You Start

- The membership must be **active**.
- The membership must have already **started** (its start date is today or in the past). You cannot switch a membership whose start date is still in the future.
- The tier you switch to must be the **same type** as the current membership. An individual membership can only switch to another individual tier, and an organization membership only to another organization tier.

## How to Switch

1. In the WordPress admin, open the membership you want to change from its member edit screen.
2. Click **Manage Membership**.
3. In the window that opens, choose **Switch Membership**.
4. Choose **Create Membership**.
5. Search for and select the tier you want to switch to. Only tiers matching the current membership's type appear in the list.
6. Click **Switch Membership** to complete the change.

When it finishes, the page opens the new membership so you can confirm everything looks correct.

## What Happens

- A new membership is created on the tier you chose.
- The **end date stays the same** as the original membership.
- The grace period follows the **new tier's** settings, so the expiry date may shift slightly if the new tier has a different grace period.
- The original membership is **cancelled** immediately.

## Tips and Warnings

- The switch takes effect **right away**.
- Switching is **not reversible in one click.** To undo it you would switch back to the original tier, which cancels the new membership and creates another. Confirm the target tier before completing the switch.
- If the tier you want does not appear in the list, check that it is the **same type** (individual or organization) as the membership you are switching.
- If the **Manage Membership** button will not open, check that the membership is active and has already started.
