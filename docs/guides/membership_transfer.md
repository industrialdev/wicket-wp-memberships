# Transfer a Membership to a New Owner

Transferring a membership moves it from its current owner to a different person. The new owner receives a fresh membership that keeps the remaining time from the original, and the original owner's membership is cancelled.

Use this when a membership needs to belong to someone else — for example, a role changes hands or the wrong person was set as the owner.

## Before You Start

- The membership must be **active**.
- The membership must have already **started** (its start date is today or in the past). You cannot transfer a membership whose start date is still in the future.
- You need the name or details of the person who will become the new owner. They must already exist in the system so you can search for them.

## How to Transfer

1. In the WordPress admin, open the membership you want to transfer from its member edit screen.
2. Click **Manage Membership**.
3. In the window that opens, choose **Transfer Membership**.
4. Start typing the new owner's name in the search box (type at least three characters), then select them from the list.
5. Review the confirmation notice and tick the box to confirm you understand the change.
6. Click to complete the transfer.

When it finishes, the page opens the new owner's membership so you can confirm everything looks correct.

## What Happens

- The new owner gets a new membership that carries the **same remaining term** (end date) as the original.
- The original owner's membership is **cancelled** immediately.
- Any linked subscription is moved to the new owner, so future renewals belong to them.

## Tips and Warnings

- **The original order (payment) stays with the person who paid.** Only the membership and its subscription move to the new owner. This is intentional — do not try to change the order to "fix" this.
- Transferring is **not reversible in one click.** To undo it you would have to transfer the membership back, which cancels the new one and creates another. Double-check the new owner before confirming.
- If the **Manage Membership** button will not open, check that the membership is active and has already started — those are the two most common reasons it is blocked.
